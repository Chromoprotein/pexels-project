## Pexels API project

Searching stock photos with the Pexel API. It has a simple Node.js backend and pagination.

### Directory

- client/src/

    * components: small reusable components

    * hooks: image fetching hooks

    * pages: page components

    * styles: the css file

- server/server.js: the server
